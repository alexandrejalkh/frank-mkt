## 3. Carl Jung 12 archetypes

### 3.1 Origem + adaptation

```
1919: CARL JUNG (Swiss psychologist)
   12 archetypes universal personality patterns
   Innate, hereditary
   Transcend language + culture + time

EARLY 2000s: MARGARET MARK + CAROL S. PEARSON
   "The Hero and the Outlaw" (2001)
   Adapted Jung's archetypes to business
   Foundation of modern brand archetype theory
```

### 3.2 Os 12 archetypes — overview

```
INNOCENT — pure, optimistic, simple, moral
   Examples: Coca-Cola, Dove, Aveeno
   Goal: be happy
   Fear: doing something wrong/bad
   Strategy: do things right

EVERYMAN — relatable, friendly, down-to-earth
   Examples: IKEA, Target, Levi's, Home Depot
   Goal: belong, connect
   Fear: standing out, being excluded
   Strategy: develop ordinary virtues

HERO — courageous, bold, inspiring
   Examples: Nike, FedEx, BMW, Disney
   Goal: prove worth via brave acts
   Fear: weakness, being vulnerable
   Strategy: become as strong + competent as possible

OUTLAW — rebel, disruptor, anti-establishment
   Examples: Harley-Davidson, Liquid Death, Virgin
   Goal: revolution, break rules
   Fear: powerlessness, being conventional
   Strategy: disrupt, destroy, shock

EXPLORER — adventurous, free-spirited, ambitious
   Examples: Patagonia, North Face, Jeep, Red Bull
   Goal: freedom, self-discovery
   Fear: getting trapped, conformity
   Strategy: journey, seek, experience

CREATOR — innovative, imaginative, artistic
   Examples: Apple, Lego, Adobe, Crayola
   Goal: create things of enduring value
   Fear: mediocre, vision unrealized
   Strategy: develop artistic skill + imagination

RULER — authoritative, leader, premium
   Examples: Rolex, Mercedes-Benz, Microsoft
   Goal: control, prosperity, success
   Fear: chaos, being overthrown
   Strategy: exercise power

MAGICIAN — visionary, transformer, charismatic
   Examples: Disney, Tesla, Polaroid, Dyson
   Goal: make dreams come true
   Fear: unintended consequences
   Strategy: develop a vision + live by it

LOVER — passionate, sensual, intimate
   Examples: Chanel, Victoria's Secret, Häagen-Dazs
   Goal: be in relationship with people, work, surroundings
   Fear: being alone, unwanted, unloved
   Strategy: become more attractive + emotionally available

CAREGIVER — empathetic, compassionate, nurturing
   Examples: Johnson & Johnson, UNICEF, Volvo, Campbell's
   Goal: care for others
   Fear: selfishness, ingratitude
   Strategy: do things for others

JESTER — playful, irreverent, fun-loving
   Examples: Old Spice, Skittles, Dollar Shave Club, Geico
   Goal: have a great time + lighten the world
   Fear: being bored, boring others
   Strategy: play, make jokes, be funny

SAGE — wise, thoughtful, expert
   Examples: Google, Harvard, BBC, The Economist, McKinsey
   Goal: find truth, understand world
   Fear: being deceived, misled, ignorance
   Strategy: seek information + knowledge
```

### 3.3 Decision matrix archetype

```
PERGUNTAS PARA ESCOLHER

1. WHO is your customer? (psychographic)
   Innocent: optimistic moral consumers
   Everyman: relatable middle-class
   Hero: ambitious self-improvers
   Outlaw: rebels, anti-establishment
   Explorer: adventurers, individualists
   Creator: artists, innovators
   Ruler: status-driven, premium
   Magician: visionaries, transformers
   Lover: emotional, sensual
   Caregiver: nurturers, parents
   Jester: fun-seekers, light-hearted
   Sage: knowledge-seekers, intellectuals

2. WHAT is your core promise?
   Each archetype delivers different promise

3. WHO are competitors?
   Identify gaps (avoid crowded archetype space)

4. WHAT is founder's authentic personality?
   Don't force archetype that doesn't match

5. CAN you sustain 5-10+ years?
   Liquid Death (Outlaw) sustained 7+ years
   Mailchimp (Jester) sustained 10+ years
```

## 4. Brand pyramid — Aaker + Keller CBBE

### 4.1 Aaker brand equity (1991)

```
DAVID AAKER FRAMEWORK

5 DIMENSIONS (asset-focused)

1. BRAND AWARENESS
   Recognition + recall
   "Have you heard of [brand]?"

2. BRAND ASSOCIATIONS
   What customers associate brand with
   Volvo = safety
   BMW = driving pleasure

3. PERCEIVED QUALITY
   Customer perception (NOT actual quality)
   Premium pricing power

4. BRAND LOYALTY
   Repeat purchase + advocacy
   Lower CAC, higher LTV

5. PROPRIETARY ASSETS
   Trademarks, patents, distribution
   Defensible moats
```

### 4.2 Keller CBBE pyramid (1993)

```
KEVIN LANE KELLER — CUSTOMER-CENTRIC

4 STAGES + 6 BUILDING BLOCKS

STAGE 4: BRAND RELATIONSHIPS (top)
   ┌─────────────────────┐
   │ Block 6: RESONANCE  │ ← deep loyalty + community
   └─────────────────────┘

STAGE 3: BRAND RESPONSE
   ┌─────────────────────┐
   │ Block 4: JUDGMENTS  │ ← rational evaluation
   └─────────────────────┘
   ┌─────────────────────┐
   │ Block 5: FEELINGS   │ ← emotional response
   └─────────────────────┘

STAGE 2: BRAND MEANING
   ┌─────────────────────┐
   │ Block 2: PERFORMANCE│ ← functional benefits
   └─────────────────────┘
   ┌─────────────────────┐
   │ Block 3: IMAGERY    │ ← psychological benefits
   └─────────────────────┘

STAGE 1: BRAND IDENTITY (base)
   ┌─────────────────────┐
   │ Block 1: SALIENCE   │ ← awareness
   └─────────────────────┘

GOAL: build customer up the pyramid
   Salience → Performance + Imagery →
   Judgments + Feelings → Resonance
```

### 4.3 Aaker vs Keller — quando usar

```
AAKER (asset-focused)
   USE FOR
   - Portfolio metrics
   - M&A valuation
   - Brand financial assets
   - Investor reporting
   - Trademark/IP management

KELLER (customer-centric)
   USE FOR
   - Brand audit (where weak/strong)
   - Brand-building strategy
   - Diagnostics (what to fix)
   - Customer relationships focus
   - Marketing campaign design

MANY ORGS USE BOTH
   Aaker = portfolio + assets
   Keller = build logic + customer diagnostics
```

### 4.4 CBBE building blocks deep dive

```
BLOCK 1: SALIENCE (awareness)
   Brand recognition
   Brand recall (top-of-mind)
   "When you think of [category], what brand?"

   Build via: distribution, advertising frequency, distinctive assets

BLOCK 2: PERFORMANCE (functional)
   Product/service capabilities
   Reliability, durability, design
   Customer service quality

   Build via: product excellence, customer experience

BLOCK 3: IMAGERY (psychological)
   User imagery (who uses brand)
   Usage imagery (when/where used)
   Brand personality + values
   History + heritage

   Build via: storytelling, content, positioning

BLOCK 4: JUDGMENTS (rational)
   Quality (perceived)
   Credibility (expertise + trust)
   Consideration (in shortlist?)
   Superiority (vs competitors)

   Build via: third-party validation, case studies

BLOCK 5: FEELINGS (emotional)
   Warmth (comfort)
   Fun (excitement)
   Excitement (energy)
   Security (trust)
   Social approval (status)
   Self-respect (pride)

   Build via: brand voice + emotional storytelling

BLOCK 6: RESONANCE (top)
   Behavioral loyalty (repeat purchase)
   Attitudinal attachment (love)
   Sense of community (belonging)
   Active engagement (advocacy)

   Build via: community, customer success, sustained brand voice
```

## 5. Brand voice + tone + personality

### 5.1 Distincao critica

```
PERSONALITY = WHO YOU ARE
   Brand's character (innate)
   Stable identity over time
   Examples: playful, authoritative, caring

VOICE = HOW PERSONALITY SHOWS UP IN WRITING/SPEAKING
   Consistent expression of personality
   NEVER changes
   Examples: casual + witty (Mailchimp), precise + technical (Stripe)

TONE = HOW VOICE ADJUSTS TO CONTEXT
   Situational variation
   Maintains voice but flexes for situation
   Examples: same voice but warmer for support, more direct for terms

ANALOGY (Mailchimp's framework)
   Personality = WHO
   Voice = WHAT (consistent)
   Tone = HOW (varies)
```

### 5.2 4-dimension framework (Nielsen Norman Group)

```
4 DIMENSIONS to position any brand voice

DIMENSION 1: FORMAL ←→ CASUAL
   Stripe = formal
   Liquid Death = casual

DIMENSION 2: SERIOUS ←→ FUNNY
   Apple = serious
   Wendy's = funny

DIMENSION 3: RESPECTFUL ←→ IRREVERENT
   Volvo = respectful
   Liquid Death = irreverent

DIMENSION 4: ENTHUSIASTIC ←→ MATTER-OF-FACT
   Innocent = enthusiastic
   Stripe = matter-of-fact

= position brand on each axis
= 4D coordinate identifies unique voice
```

### 5.4 Brand voice doc — elementos

```
ELEMENTOS BRAND VOICE DOC

1. PERSONALITY 3-5 ADJECTIVES
   "Confident, warm, witty"
   NOT "innovative, leading, premium" (generic)

2. ARCHETYPE
   Primary + secondary
   Hero / Sage / Caregiver / Jester etc

3. 4-DIMENSION POSITIONING
   Formal vs casual: 7
   Serious vs funny: 4
   Respectful vs irreverent: 6
   Enthusiastic vs matter-of-fact: 5

4. VOCABULARY GUIDELINES
   WORDS WE USE: clear, concrete, honest
   WORDS WE AVOID: revolutionary, leverage, synergy

5. TONE SCALES (per context)
   Customer support: warmer, slower
   Terms/legal: direct, clear
   Sales: confident, helpful
   Crisis: honest, accountable

6. EXAMPLES (do + dont)
   ✅ "Sign up in 30 seconds"
   ❌ "Effortlessly onboard onto our intuitive platform"

7. WRITING PRINCIPLES
   Sentence length preferences
   Active vs passive voice
   Punctuation conventions
   Capitalization rules

8. PER-PLATFORM ADJUSTMENTS
   Cf. `multi-platform-narrative` Sec 11.3 + 6
   LinkedIn: professional emphasis
   Instagram: visual + concise
   TikTok: casual + bold
```

## 6. Brand identity verbal + visual

### 6.1 Verbal identity

```
COMPONENTS

1. POSITIONING STATEMENT (cf. `posicionamento-marca` Sec 6)
   25-30 words

2. TAGLINE
   3-7 words ideally
   Examples: "Just Do It" (Nike), "Think Different" (Apple),
             "Murder Your Thirst" (Liquid Death)

3. KEY MESSAGES
   3-5 core messages reused
   Hierarchy (primary + supporting)

4. BRAND VOICE DOC
   See Sec 5.4

5. NARRATIVE FRAMEWORK
   Story arc (problem + journey + resolution)
   Origin story

6. NAMING CONVENTIONS
   Product names
   Feature names
   Sub-brands

7. VOCABULARY GUIDELINES
   Use-words + avoid-words

8. WRITING PRINCIPLES
   Tone scales
   Format conventions
```

### 6.2 Visual identity (cf. `composicao-visual`)

```
COMPONENTS

1. LOGO
   Primary mark
   Secondary marks (icon, wordmark)
   Variations (color, mono, reversed)

2. COLOR PALETTE
   Primary, secondary, accent
   60-30-10 rule
   Hex/RGB/CMYK values
   Cf. `composicao-visual` Sec 2

3. TYPOGRAPHY
   Headline font + body font
   Font scale + weights
   Cf. `composicao-visual` Sec 3

4. IMAGERY STYLE
   Photography (lighting, composition)
   Illustration style
   Mood + tone

5. ICONOGRAPHY
   Icon set
   Style consistency

6. GRAPHIC ELEMENTS
   Patterns, shapes, textures
   Brand-specific motifs

7. MOTION PRINCIPLES
   Animation style
   Transition guidelines

8. DATA VISUALIZATION
   Chart styles
   Color usage in data

9. SONIC IDENTITY (cf. `audio-musica-ia` Sec 2)
   Audio logo
   Sonic mnemonic
   Brand voice (audio)

10. MASCOT (if applicable)
    Cf. `composicao-visual` Sec 9
    Duolingo Duo, Mailchimp Freddie, Magazine Luiza Lu
```
