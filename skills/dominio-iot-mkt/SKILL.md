## 2. Fundacao 2 — M5 Stack ESP32 inside (user-mentioned focus)

**O que é M5 Stack** (m5stack.com):
- **Open-source modular toolkits IoT** baseados ESP32.

- **User-friendly IDE** — UIFlow + Arduino + ESP-IDF + MicroPython.
- **Prototipagem rapida + product-ready solutions**.
- Aplicacoes: **education + smart interfaces + industrial HMI + IoT product development**.

### 2.2. M5 Stack product lines principais 2026

**M5Stack Core2** (flagship developer board):
- ESP32-D0WDQ6-V3 chip.
- **Dois cores Xtensa 32-bit LX6** independentes.
- Frequência **240 MHz**.
- **Wi-Fi integrated**.
- **2.0-inch capacitive touch screen** colorida.
- I/O extensiva (GPIO + I2C + SPI + UART).
- Battery + speaker + microphone integrated.

**M5Stack Tough** (industrial-grade):
- Espressif ESP32 main control.
- **Dual-core LX6 240 MHz**.
- **Fully enclosed casing** (dustproof + waterproof IP67).
- Aplicacao: **industrial field control + smart buildings + outdoor data acquisition**.
- Robust para ambientes adversos.

**M5Stack Basic Core**:
- ESP32 chip standard.
- **16 MB FLASH memory**.
- **2.0-inch IPS display** integrated.
- **Speaker + TFCard slot**.
- Entry-level prototipagem.

**ATOM series** (compact):
- Smaller form factor.
- Embedded use cases.
- Lower cost.

**Stick series** (portable):
- Stick form factor.
- Wearable applications.
- Compact battery integrated.

**Option 1 — Arduino IDE** (most popular):
