## 1. Visao Geral

A regra-chave 2026: **uma mensagem nao e uma peca de conteudo replicada em N lugares — e UMA estrategia expressa em N comportamentos diferentes.** Cross-post identico = **SUPPRESSED** em todas plataformas; native content = **boosted**. Hub-and-spoke (1 pillar + N spokes) gera **+35% reach** vs isolated posts e **+40% output** sem aumentar tempo.

Esta skill cobre 6 fundacoes:

1. **Cenario 2026** — algoritmos punem cross-post identico universalmente.

4. **Aspect ratios + length** — 9:16 gold standard mobile 2026 (95% consumption).
5. **Tom + voice por plataforma** — LinkedIn formal vs TikTok casual.
6. **Sequencing temporal** — simultaneous / waterfall / cascade.

Mais 2 aplicacoes:

7. **Workflow operacional** — pillar production → atomization → distribution.
8. **AI tools 2026** — Opus Clip, Repurpose.io, Castmagic, Riverside.

   - Mais tolerante (long-form mainly)

## 3. Atomic content vs Repurposing

### 3.1 Diferenca conceitual

```
REPURPOSING
   Pegar pillar e REFORMATAR para novo formato
   Ex: blog 3000 palavras → video YouTube 10min (mesmo conteudo, formato diferente)

ATOMIZATION
   QUEBRAR pillar em pecas INDEPENDENTES
   Ex: blog 3000 palavras → 30 standalone posts
   - 5 LinkedIn carousels
   - 8 Twitter threads
   - 10 Instagram quotes
   - 5 TikTok shorts
   - 2 newsletters
   = cada peca standalone, vivendo sua propria vida

QUANDO REPURPOSING
   Quando pillar funciona em formato diferente sem perder essencia
   Ex: webinar → blog post (transcript + edit)

QUANDO ATOMIZATION
   Quando pillar tem N insights independentes
   Ex: research report 50 paginas → 30 mini-insights
```

### 3.2 Pillar content sources

```
TIPOS DE PILLAR (rich em ideias)

A) WEBINAR / WORKSHOP (45-60min)
   = transcript +  insights + Q&A
   → 20-30 atomized pieces

B) PODCAST EPISODE (30-45min)
   = audio + transcript + quotes
   → 15-25 pieces

C) LONG-FORM ARTICLE (3000+ palavras)
   = ja text-first
   → 10-15 pieces

D) RESEARCH REPORT (PDF 20-50p)
   = data + visualizations + quotes
   → 30-50 pieces

E) FOUNDER TALK / CONFERENCE
   = video + transcript + slides
   → 20-30 pieces

F) CUSTOMER INTERVIEW (30min)
   = quotes + case study + video
   → 10-15 pieces
```

### 3.3 1 pillar → 30+ pieces matrix

EXEMPLO: PODCAST 30MIN COM CEO

ATOMIZATION OUTPUTS

## 4. Hub-and-spoke model

### 4.1 Estrutura

```
              [PILLAR CONTENT]
                    |
       ____________|____________
      |       |    |     |     |
   LINKEDIN  IG   TT     X    YT     EMAIL
      |       |    |     |     |     |
   carousel reels short thread shorts newsletter
   article  story        long      
   text     posts  
```

### 4.2 Workflow hub-and-spoke

```
1. PILLAR PRODUCTION (1 vez)
   - Create high-quality long-form (3-8h)
   - Aprovar em comite (se org)
   - Brand voice consistente

2. ATOMIZATION (4-8h)
   - Identificar 30+ insights/quotes
   - Mapear para plataformas
   - AI tools (Opus Clip, Castmagic) ajudam

3. ADAPTATION POR PLATAFORMA (8-16h)
   - Re-write captions tom-especifico
   - Re-format aspect ratios
   - Adicionar native elements (sounds, effects)
   - Add captions accessibility

4. SCHEDULING (1-2h)
   - Calendar 2-4 semanas
   - Native scheduler tool (NAO cross-post automated)
   - Best times por plataforma

5. ENGAGEMENT (ongoing 30min/post)
   - Reply primeiros comentarios <1h
   - Cross-pollinate ("conversa em [plataforma]")
   - Iterate baseado em performance
```

### 4.3 Cadencia recomendada

```
PILLAR
   1 a cada 2 semanas (B2B) ou 1 a cada semana (creator)

SPOKES
   LinkedIn: 3-5 posts/sem
   Instagram: 3-5 reels/sem + stories diarias
   TikTok: 5-7 videos/sem (volume = key)
   X: 3-7 tweets/dia
   YouTube long: 1-2 videos/sem
   YouTube Shorts: 3-5/sem
   Newsletter: 1/sem
```

## 5. Aspect ratios + length 2026

### 5.1 9:16 gold standard 2026

```
95% MOBILE VIDEO CONSUMPTION

9:16 (1080 × 1920) = MOBILE VERTICAL
   - TikTok primary
   - Instagram Reels primary
   - YouTube Shorts primary
   - LinkedIn vertical (preferential placement 2026!)
   - Facebook Stories/Reels
   - Snapchat
   = grava SEMPRE 9:16 se tem chance de viralizar mobile

16:9 (1920 × 1080) = LANDSCAPE
   - YouTube long-form (default)
   - LinkedIn landscape (legacy)
   - TV ads
   - Webinar recordings
   - Tutorials desktop

4:5 (1080 × 1350) = MOBILE PORTRAIT
   - Instagram Feed (preferential)
   - Facebook Feed
   = ocupa mais real estate em mobile feed scroll

1:1 (1080 × 1080) = SQUARE
   - Instagram Feed (legacy)
   - LinkedIn Feed (compact)
   = funciona OK mas inferior a 4:5 mobile
```

### 5.2 Strategy de gravacao otimizada

```
GRAVAR 1 VEZ, USAR EM N FORMATOS

OPCAO A — GRAVAR 9:16 NATIVO
   Pros: optimal mobile (95% consumption)
   Cons: dificil cropp para landscape
   Use: creator content, quick produces

OPCAO B — GRAVAR 16:9 + CROP
   Pros: flexivel (crop 9:16, 4:5, 1:1)
   Cons: 9:16 sai com elementos cortados
   Use: webinars, formal interviews

OPCAO C — GRAVAR MULTI-CAM 16:9 + 9:16
   Pros: max flexibility
   Cons: production cost
   Use: enterprise content high-budget

RECOMENDACAO 2026
   Si volume + speed = 9:16 nativo
   Si formal + multi-purpose = 16:9 + crop com Opus Clip
```

```

   21-34s = storytelling
   30-60s = educational
   60s-3min = deep dive (B2B B2C tutoriais)

   60-90s = sweet spot 2026 (was 30s em 2024)

   30-45s = sweet spot
   60s = max

YOUTUBE LONG
   8-15min = sweet spot 2026 (was 10-12min)
   30-60min OK para tutorials/podcasts

LINKEDIN VIDEO
   1-3min = native video sweet spot
   30-60s sufficient para hooks
   30-90s LinkedIn Lives clips
   Document carousels: 7-10 slides

X
   2-3min video max effective
   140 chars hook + thread 8-12 tweets

FACEBOOK
   1-3min Reels
   3-5min long video posts (community)

PODCAST
   25-45min = sweet spot
   60-90min OK se autoridade alta
```

## 7. Sequencing temporal

### 7.1 3 estrategias

```
A) SIMULTANEOUS — push 48h concentrado
   Plataformas: HN + Reddit + PH + X + LinkedIn
   Use: launch open-source, product launch
   Beneficio: velocity → Trending algorithm
   Cf. `github-presence` Sec 7

B) WATERFALL — single by single
   Plataformas: 1 plataforma por dia/semana
   Use: research report rollout, course launch
   Inspirado em Spotify EP waterfall
   Beneficio: 5 chances de descoberta vs 1

C) CASCADE — janela 24-48h por plataforma
   Plataformas: blog → email → LinkedIn → IG → TikTok
   Use: thought leadership, content rollout normal
   Beneficio: cada plataforma respira sem competir
```

```

   ☐ Blog post live

   ☐ Reddit r/[niche]

   ☐ Instagram Reel
   ☐ TikTok video

HOUR +6 to +24
   ☐ YouTube Shorts

HOUR +24 to +48

```

WEEK 1 — Cap 1

```

   8h: gravar pillar (podcast/webinar/long video)
   2h: edit basico
   = 10h investimento 1 vez/semana

TERCA — ATOMIZATION
   2h: AI tools (Opus Clip, Castmagic) extract clips
   2h: identify quotes/insights
   2h: outline 30+ pieces
   = 6h

QUARTA — ADAPTATION TEXT
   4h: write captions LinkedIn + X + IG
   2h: blog post (3000 palavras com transcript)
   = 6h

QUINTA — ADAPTATION VIDEO
   4h: edit clips por plataforma
   2h: aspect ratio re-frame
   1h: captions accessibility
   = 7h

SEXTA — ADAPTATION IMAGE + SCHEDULING
   3h: quote cards Canva
   2h: scheduling (native tools por plataforma)
   1h: review cadencia
   = 6h

SABADO — DOMINGO
   Ongoing: respond comments primeiras 24h apos posts

TOTAL SEMANAL: 35h (1 produtor + 1 editor)
OUTPUT: 30-50 pieces of content publicados em 2-4 semanas
```

## 10. Anti-patterns

| **Tom wrong** (TikTok dance em LinkedIn) | Brand mismatch confusing |

| **Mismo CTA todas plataformas** | LinkedIn click-out vs TikTok save sao diferentes |

## 12. Regras de Ouro

1. **NAO CHUTAR** — cross-post identico = penalty universal. Native content always.
2. **Hub-and-spoke +35% reach** — pillar 1 + N spokes vs isolated.
3. **9:16 = gold standard 2026** — 95% mobile video consumption.
4. **Captions accessibility obrigatorio** — 75% watch sound off; -30% reach sem.
5. **Tom adaptado por plataforma** — LinkedIn formal vs TikTok casual.
6. **Length por plataforma** — TikTok 11-18s viral, Reels 60-90s sweet, YouTube long 8-15min.
7. **Sequencing escolhido** — simultaneous (launch) vs waterfall (capitulos) vs cascade (normal).
8. **Native scheduler** por plataforma — NAO Buffer simple cross-post.
9. **AI tools repurposing** (Opus Clip + Castmagic + ChatGPT) economizam 60-70% tempo.
10. **Brand voice doc** documentado per-platform — consistencia + adaptacao.
11. **Atomization > Repurposing** quando pillar tem N insights independentes.
12. **+40% output sem aumentar tempo** com hub-and-spoke disciplinado.
13. **Pre-launch 6-8 sem + launch 1-2 sem + post-launch 4-6 sem** — 3 phases.
14. **Watermark de outras plataformas** = penalty. Re-export sem.
15. **Hashtags por plataforma** — IG 3-5, TikTok 3-5, X 1-2, LinkedIn 3-5, YouTube tags.
16. **Best times locais por plataforma** — NAO mesmo time todas.
17. **Engagement primeiras 24h** apos cada post — algoritmo critica.
18. **Iterate baseado em performance** — analytics nativos por plataforma.

### Exemplo 2 — "Cross-post identico Buffer — diagnostico penalty"

Quanto reach perdeu vs sem watermark; (3) **Caption analysis**: same caption hashtags todas plataformas (sinal cross-post)? (4) **Timestamp analysis**: posts identico timestamp todas plataformas (sinal automation)? **Plano fix 8 semanas**: **WEEK 1-2 — Stop bleeding**: cancelar Buffer simple cross-post + comecar native posting (LinkedIn native + IG native + TikTok native diferentes app), schedule individual por plataforma; **WEEK 3-4 — Native content production**: comecar pillar weekly + atomization (cf. workflow Sec 8 desta skill), invest em Opus Clip + Castmagic ($48/mes total); **WEEK 5-6 — Tom adaptation**: documentar brand voice per-platform (template Sec 11.3), re-write captions com tom plataforma-especifico, ChatGPT $20 para velocity adaptation; **WEEK 7-8 — Recovery + iterate**: monitor analytics, expect reach recovery 50-100% em 60-90 dias (nao instant — algorithms aprendem mas com lag), iterate baseado em best performers per platform. **ROI esperado**: reach atual presumido 1k impressions/post LinkedIn (decadente) → 3-5k impressions com native + adaptado em 90 dias = **+200-400% recovery**. **Investment**: 35h/sem 1 produtor + 1 editor (vs 10h/sem Buffer + posting cross-post — = +25h/sem para production native, mas reach 3-5x = ROI clearly positive). **Tools Stack recomendado**: Opus Clip $19 + Castmagic $29 + ChatGPT $20 + Canva $13 + native schedulers free (Meta Business Suite, LinkedIn native) = $81/mes. **Caveat**: NAO esperar fix imediato — algorithms tem memoria 60-90 dias de cross-post pattern; recovery gradual. **Anti-padroes alvo**: continuar Buffer simple cross-post, manter watermarks, hashtags transversais, mesmo timing todas plataformas. **Quick win imediato (1 sem)**: cancelar Buffer cross-post + 1 native post LinkedIn de qualidade alta = sinal positivo algoritmo.

**Comportamento-chave:** diagnostico causa-raiz universal cross-platform, plano 8 semanas com ROI projection, tools stack $81, caveat sobre recovery nao instantaneo, quick win imediato.
