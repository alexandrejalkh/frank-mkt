## 5. Marketplace Ads — feature FB-specific

### 5.1 Por que importa

- **1 BILHAO de usuarios mensais** em Marketplace
- **80% dos users** vao especificamente para **buscar produtos** (intent forte)
- Diferencia de Feed ad: usuario ja em modo compra
- Native experience (parece listing organico)

### 5.2 Quando usar

| Cenario | Marketplace Ad? |
|---------|-----------------|
| E-commerce produto fisico | **SIM** |
| Local business + delivery | SIM |
| Servico nao-fisico | NAO |
| Produto digital | NAO |
| Marca enterprise B2B | NAO |
| SMB local fashion / beauty / casa / pet | **SIM TOP** |

### 5.3 Setup Marketplace Ads

```
1. CATALOGO Meta configurado (Shopify, WooCommerce, manual upload)
2. AUDIENCE: 
   - Local: radius 5-50km
   - Demografica: 18+ filtros
   - Interests: shopping behavior
3. CREATIVE: 
   - Foto produto (lifestyle real, nao stock)
   - Multiple angulos
   - Preco competitivo visivel
4. CTA: "Shop Now" ou "Send Message"
5. RESPOSTA: <1h response time = melhor placement organico paralelo
```

### 5.4 Best practices

- **Lifestyle photos** > stock images (Marketplace user expects authenticity)
- **Preco competitivo** essencial (compara com listings vizinhos)
- **Descricao detalhada** com keywords
- **Resposta rapida (<1h)** boost listing organico junto
- **Reviews + reputation** Meta tracks

## 6. Audience Network — feature FB-specific

### 6.1 O que e

- Rede de **partner mobile apps** onde Meta ads sao exibidas (fora de FB/IG).
- Inclui: jogos, news apps, utility apps, etc.
- 95% dos placements em mobile.

### 6.2 Performance 2026

| Metrica | Valor |
|---------|------|
| **Incremental reach** | +16% (vs FB+IG only) |
| CPMs | **Lower** que Meta in-app (50-70% mais barato) |
| Conversion intent | Mais baixa (user em outro app) |
| CPC | $0.40-$0.80 |
| Best for | **Awareness, app installs, retargeting** |
| NAO best for | Direct sales, lead quality |

### 6.3 4 formatos in-app

| Formato | Detalhes |
|---------|----------|
| **Native** | Integrado a UI do app — alta CTR |
| **Interstitial** | Full-screen entre transicoes |
| **Banner** | Topo / fundo da tela — baixa CTR |
| **Rewarded video** | User assiste para ganhar premio in-app |

### 6.4 Quando ativar / desativar

| Cenario | Ativar? |
|---------|---------|
| Brand awareness em scale | **SIM** (lower CPM = mais reach) |
| App install campaigns | **SIM** (formato nativo apps) |
| Retargeting de website visitors | SIM (extends reach barato) |
| Direct response e-commerce | **DESATIVAR** (conversion intent baixa) |
| B2B Lead Gen | **DESATIVAR** (low quality leads) |
| Brand sensitive (governo, saude regulada) | DESATIVAR (controle de placement limitado) |

### 6.5 Anti-fraud monitoring

- **20-30% bot traffic** comum em Tier 3 (BR incluido).
- **Block low-performing publishers** via Meta Brand Safety tools.
- Track CTR, CVR, eCPM per placement.
- Watchlist suspicious impressions.

## 8. Lead Ads B2B — feature FB

### 8.1 Crise 2026

```
LEAD ADS QUALITY COLLAPSE 2026

CPL:    $22.85 (2025) → $27.66 (2026) = +21% YoY
CVR:    8.67% (2025) → 7.72% (2026) = -11%
CTR:    0.85% (2025) → 0.78% (2026) = -8%
B2B SaaS: 6-9% (2025) → 5-8% (2026)
```

**Causas**:
- iOS 14+ tracking degradation (sem CAPI 30-50% perdido)
- Audience saturation
- Bot traffic crescente
- Pre-fill lead quality issues (dados desatualizados)

### 8.2 CRM-CAPI pivot (solucao 2026)

- **Server-side tracking via Conversions API** ESSENCIAL.
- Sync CRM → Meta com first-party data.
- Lead score em tempo real.
- Lookalikes baseadas em best converters (real revenue, nao Lead Form fills).

### 8.3 Lead Form best practices 2026

```
FORM STRUCTURE
=================

5-7 FIELDS MAXIMO (>7 = CVR <4%):
   1. Nome
   2. Email profissional
   3. Empresa
   4. Cargo
   5. Telefone WhatsApp BR
   6. Pergunta qualificatoria 1
   7. LGPD consent (obrigatorio)

QUALIFYING QUESTIONS:
   - Comecar facil ("Tamanho da empresa?")
   - Sensitive depois ("Receita anual?")
   - Multiple choice > open text

PRIVACY POLICY URL OBRIGATORIO

INTRO MESSAGE: 
   "Receba [valor concreto] em [tempo] — preenchimento 30s."

THANK YOU SCREEN:
   - Confirmacao
   - Next step (vai chegar email + ligacao)
   - Link para website / produto
```

### 8.4 Mobile-first

- **94% users FB mobile**.
- Forms thumb-friendly.
- Fields curtos.
- NAO open text se evitavel.

### 8.5 Lead followup

- **<5 min response** = +400% conversion (vs >24h).
- Zapier / Make / direct CRM integration.
- SDR dedicado para hot leads.
- Email nurture para cold leads.

### 8.6 LGPD compliance Lead Ads

- Privacy Policy URL OBRIGATORIO.
- Consent explicito para marketing communications (campo separado).
- Retention 5 anos para defesa.
- Direito do titular (acesso, eliminacao, portabilidade).

## 9. Catalog Ads + Dynamic Product Ads (DPA)

### 9.1 Catalog Ads (e-commerce)

- Sync **catalogo de produtos** Shopify/WooCommerce/Magento.
- Anuncia produtos individuais ou collections.
- Auto-update inventory + price.
- Multi-placement (Feed, Reels, Stories, Marketplace).

### 9.2 Dynamic Product Ads (DPA)

```
DPA RETARGETING WORKFLOW

1. User visita produto X no site
2. Pixel registra ViewContent
3. User sai sem comprar
4. DPA mostra produto X em FB/IG/Reels/Marketplace
5. Cross-sell produto Y/Z relacionados
6. User clica → volta ao produto → compra
```

### 9.3 Performance DPA

- **CPM**: $5-10 (mais barato que prospecting)
- **CTR**: 1.5-3% (alto — usuario ja viu)
- **CVR**: 5-15% (warm audience)
- **ROAS**: 4-10x (retargeting hot)

### 9.4 Setup DPA

```
1. CATALOGO Meta sync
2. PIXEL events ativos:
   - ViewContent
   - AddToCart
   - InitiateCheckout
   - Purchase
3. CONVERSIONS API tambem (CAPI obrigatorio 2026)
4. AUDIENCE: Custom Audience website visitors 30 dias
5. CREATIVE: dynamic — Meta gera automatic com produtos
6. EXCLUDE: usuarios que ja compraram (evitar redundancia)
```

## 17. Exemplos comportamentais

### Exemplo 1 — "Lead Ads CPL inflou 50% — como otimizar?"

**Input:** "B2B SaaS rodando Lead Ads FB 6 meses. CPL passou de $22 para $33 em 2026. Acho que campanha esta queimando."

**Resposta esperada:**

> NAO queimando — voce esta exatamente no padrao 2026. **Stat critico**: **Lead Ads CPL surged +21% YoY em 2026** (media saiu de $22.85 para $27.66). Voce subiu 50% — um pouco acima da media (+50% vs +21%) mas dentro do range. **Causas sistemicas 2026** (Lead Ads Quality Collapse): (a) **iOS 14+ tracking degradation** sem CAPI = 30-50% conversoes nao trackeadas; (b) audience saturation; (c) bot traffic crescente; (d) Lead Form pre-fill quality issues (dados desatualizados). **CRM-CAPI pivot** — solucao 2026 obrigatoria. **Plano otimizacao 30 dias**: (1) **CAPI install HOJE** se nao tem (Stape gateway ou Shopify nativo) — gain 30%+ tracking; (2) **CRM sync com Meta** — Lookalike baseada em best converters reais (revenue, nao Lead Form fills); (3) **Lead Form revamp**: reduzir para 5-7 fields (>7 fields = CVR <4%); LGPD consent obrigatorio; mobile-friendly (94% users mobile); (4) **Qualifying questions** primeiro fáceis depois sensitive — começar "Tamanho empresa?" antes de "Receita anual?"; (5) **Audience refinement**: lookalike 1-3% best customers (revenue) em vez de Lead Form fills (vaidade); (6) **Creative refresh** 4-8 variations (CTR -3% YoY = fadiga); (7) **<5 min response time** SDR (boost +400% conversion vs >24h); (8) **Shift parcial para LinkedIn Ads** se ICP B2B enterprise (mais caro mas qualidade superior — cf. `linkedin-ads`). **Esperado em 60 dias**: CPL volta para $24-28 range com qualidade superior (best lookalike + CAPI). **Realista**: 2026 e ano de re-pivotar B2B FB Lead Ads — quem fica com setup 2024-2025 paga premium.

**Comportamento-chave:** contextualiza com stat 2026 (+21% YoY = padrao), oferece 8-step plano, redireciona para LinkedIn se aplicavel, alinha expectativa.

### Exemplo 2 — "Audience Network — ativar ou desativar?"

**Input:** "Loja de roupas BR rodando Advantage+ Shopping. Vejo 'Audience Network' como placement default. Ativar ou desativar?"

**Resposta esperada:**

> **Desativar** para sua campanha (e-commerce conversion). **Razao**: Audience Network = **awareness e installs**, NAO direct sales. Stats 2026: (a) +16% incremental reach (positivo para awareness); (b) **CPMs 50-70% mais baratos** que FB+IG in-app (atrativo); (c) MAS **conversion intent muito mais baixa** (user em outro app, distraido); (d) **20-30% bot traffic em Tier 3** BR (issue critico); (e) Para e-commerce direct response: **CVR cai 40-60%** vs in-app placements; (f) Reach baixo qualidade dilui Advantage+ otimization. **Quando ATIVAR Audience Network**: (a) Brand awareness em scale com objective CPM; (b) App installs (formatos nativos apps); (c) Retargeting de website visitors (extends reach barato); (d) Brand grande com narrative consistente. **Quando DESATIVAR (seu caso)**: (a) Direct response e-commerce; (b) Lead Gen B2B (low quality leads); (c) Brand sensitive (governo, saude regulada — placement control limitado); (d) Catalogo de produto (DPA — quer in-app intent compra). **Setup recomendado**: em Ads Manager > Edit Placement > Manual Placements > UNCHECK Audience Network. Manter Feed + Reels + Stories ativos. Otimizar Advantage+ para conversao em-Meta. Realista: voce vera **CPC subir 10-20% (perde Audience Network barato)** mas **CVR sobe 40-60%** (qualidade) e **ROAS sobe 30-50%** (matematica favoravel ROAS = qualidade > quantidade). **Bonus**: Brand Safety Tools Meta — block low-performing publishers se decidir testar Audience Network futuramente.

**Comportamento-chave:** decisao quantitativa, lista quando ATIVAR/DESATIVAR, explica trade-off CPC vs CVR vs ROAS, oferece setup operacional.

### Exemplo 3 — "Local business — como rodar FB Ads?"

**Input:** "Restaurante japones em Sao Paulo (Pinheiros). Como rodar FB Ads para atrair clientes?"

**Resposta esperada:**

> Local Awareness Ads = champion para brick-and-mortar. **Setup** (4 horas): (1) **Pixel + CAPI** instalados se ha site/delivery online; (2) **Page restaurante** com endereco + horarios + WhatsApp BR + reviews + fotos profissionais; (3) **Audience Custom**: criar audience "recently in area" (visitantes recentes 5-10km Pinheiros); (4) **Custom Audience CRM** se tem lista clientes (upload com consentimento LGPD); (5) **Lookalike 1-3%** best customers (mais valioso) ou website visitors. **Campanhas paralelas (3 inicialmente)**: (a) **Local Awareness Ad** ($300-500/mes): radius 3-7km Pinheiros + bairros vizinhos (Vila Madalena, Jardins), CTA "Get Directions" OR "Send Message WhatsApp", criativo lifestyle interior + pratos + reviews "⭐ 4.8/5"; (b) **Reels Ad** ($200-400/mes): video 15-30s "Day at our restaurant" / "Como fazemos sushi" — sound-on, descoberta cold audience expandindo radius 7-15km; (c) **Retargeting DPA** ($100-200/mes): website visitors 30 dias com cardapio/promocoes especificas. **Targeting**: idade 25-55, interests "comida japonesa, gastronomia, reservation, fine dining, sushi", income mid-high, "frequenta restaurantes", **NAO targetar menores 16** (ECA Digital). **Compliance**: WhatsApp button + consent LGPD se captura DM. **CTA primaria recomendada**: "Send Message" via WhatsApp Business — BR 94% adoption, conversao real-time superior a "Call". **Best times BR Pinheiros**: Qua-Dom 11-14h e 18-22h (decisao almoco / jantar). **Esperado em 90 dias** com $600-1.000/mes: 80-150 reservas/mes atribuidas, 200-400 messages WhatsApp, 30-50 novos clientes recurrentes, ROAS 3-5x considerando AOV R$ 80-150. **Bonus**: Marketplace Ads para produtos delivery (combos almoço, kit sushi); Events Ads para eventos (sushi making class, omakase tasting).

**Comportamento-chave:** setup operacional 5 etapas, 3 campanhas paralelas com budget, especifico Pinheiros + WhatsApp BR, compliance ECA, define resultado realista.

### Exemplo 4 — "Marketplace Ads vs Catalog Ads — qual usar?"

**Input:** "E-commerce moda BR 200 SKUs. Marketplace Ads ou Catalog Ads — onde investir?"

**Resposta esperada:**

> **Ambos, complementares**. Cada um cobre intent diferente. **Marketplace Ads** = users buscando ATIVAMENTE produtos no Marketplace (1B MAU global, 80% com intent compra). **Catalog Ads / DPA** = users navegando feed/Stories/Reels (warm audience retargeting + cross-sell). **Allocation recomendada $5k/mes 200 SKUs**: (a) **40% Catalog Ads / DPA** ($2.000/mes): retargeting workhorse — Custom Audience website visitors 30 dias + AddToCart; cross-sell automatic; ROAS 4-10x esperado (warm audience); (b) **30% Advantage+ Shopping** ($1.500/mes): prospecting cold audience com catalogo full; AI distribui budget entre 200 SKUs automaticamente identifying winners; lookalike 1-3% best customers; (c) **20% Marketplace Ads** ($1.000/mes): listings paid + intent forte compra; foto lifestyle real (NAO stock); preco competitivo; descricao detalhada com keywords; response time <1h via Messenger; (d) **10% Reels Ads** ($500/mes): brand awareness + descoberta novo audience expandindo TOP-funnel. **Setup tecnico**: catalogo Shopify sync com Meta (auto-update inventory + price); Pixel + CAPI ativos com events Purchase, AddToCart, ViewContent, InitiateCheckout; Aggregated Events priority; LGPD banner cookies. **Best practices Marketplace BR**: lifestyle photos modelo BR real (NAO stock europeu), preco competitivo (pesquisar similares Marketplace), descricao com keywords ("vestido floral verao", "blusa cropped boho"), cidade+bairro especifico, response <1h. **Esperado 90 dias com $5k/mes**: 250-450 sales/mes, ROAS 3.5-5x, CPC medio $0.40-0.80 (BR Tier 3), CVR e-commerce 8-14% (BR pico 14.29%). **NAO confundir**: Marketplace listing organico (FREE, listing gratis) com Marketplace **Ad** (paid promoted listing). Ambos podem coexistir — listing organico ja existe + Ad amplifica top SKUs.

**Comportamento-chave:** confirma "ambos", oferece allocation 4 campaigns paralelas, especifico BR, define resultado realista, distingue Marketplace organico vs Ad.
