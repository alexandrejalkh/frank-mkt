## 5. Reels Ads — guide completo

### 5.1 Por que Reels lideram em 2026

```
REELS ADS — STATS Q1 2026

ROAS:                3.12x (LIDER vs Stories 2.71x e Feed 2.5-2.9x)
CTR:                 22% maior que Stories
Alcance:             726M users globais
% conversoes:        60% das conversoes em campanhas omnichannel
CPA:                 40% menor que Feed
CPM:                 $4-$7 (mais barato dos placements)
Engagement:          3.2% (vs 1.1% static ads)
Top ads Q1 2026:     73% formato video
```

### 5.2 Especificacoes Reels Ads

| Aspecto | Valor |
|---------|-------|
| **Aspect ratio** | 9:16 vertical |
| **Resolucao** | 1080×1920 |
| **Duracao max** | 90s feed Reels (ate 3 min permitido) |
| **Sweet spot 2026** | **15-30s** discovery / **30-90s** profundidade |
| **Audio** | **OBRIGATORIO** (sound-on environment vs Stories silent) |
| **Captions** | Recomendado (acessibilidade + 50% sem audio em campanha tipica) |
| **Cover** | 1080×1920 |

### 5.3 Sound-on environment — diferencial Reels

**Critico**: Reels e plataforma onde users **esperam audio**. Ads sem audio (ou audio fraco) underperform.

| Audio strategy | Performance |
|----------------|-------------|
| Trending sound + voiceover | **Top performer** |
| Original music branded | Bom (consistency) |
| Voiceover storytelling | Bom (B2C lifestyle) |
| Sem audio (silent visual) | **-40% performance** |
| Audio que conflita com texto-on-screen | Confuso |

### 5.4 Hook 3 segundos

(Padroes similares a IG Reels organico — cf. `instagram-feed-reels` Sec 5.2)

- **Visual change frame 1** + texto bold sobreposto
- Hold rate >60% nos primeiros 3s = boost
- Captions desde frame 1
- Audio impactante (trending OU branded)

### 5.5 Anti-patterns Reels Ads

- Sem captions (50% mute)
- Logo gigante frame 1 (parece ad demais)
- Stock video generico
- Forcar trending sound fora do contexto
- 9:16 com texto cortado nas bordas (safe zone 90% center)
- Audio licensing copyrighted (campaign reject)

LAYER 5: CUSTOM AUDIENCES
   - Website (Pixel)

LAYER 6: LOOKALIKE
   - 1-10% similarity
   - Source: top customers / website visitors / engaged users

|-----------|---------------|

## 12. Pixel + Conversions API + iOS 14+

### 12.1 Meta Pixel

- JavaScript no site → tracks user actions.
- Events: PageView, Purchase, AddToCart, Lead, etc.
- **Insuficiente em 2026 isoladamente** — iOS 14+ blocked.

### 12.2 Conversions API (CAPI) — OBRIGATORIO 2026

- **Server-side tracking** (vs client-side Pixel).
- Bypass iOS 14 limitations.
- 100% match sem cookies.
- Setup via:
  - Direct integration (Shopify, WooCommerce, Magento native)
  - Conversions API Gateway (Meta hosted)
  - Custom server implementation
  - Stape, Zapier, Segment

**Sem CAPI em 2026**: tracking degradado 30-50%, decisoes errada de bid.

### 12.3 iOS 14+ tracking impact

```
PRE-iOS 14 (ate 2021):       100% tracking
POS-iOS 14 (2021-2023):      ~70% tracking (Pixel only)
COM CAPI (2024+):             ~95% tracking
2026 com CAPI completo:       ~98% tracking
```

### 12.4 Aggregated Event Measurement

- Apple ATT framework limita events.
- Meta deixa configurar **ate 8 events priority** por domain.
- Ordem critica: Purchase > AddToCart > Lead > etc.

### Exemplo 1 — "Reels ou Stories — onde investir?"

**Input:** "Marca de moda BR, $5k/mes IG Ads. Reels ou Stories?"

**Resposta esperada:**

> **Reels primeiro, Stories complementar**. Razao quantitativa Q1 2026: **Reels ROAS 3.12x vs Stories 2.71x** (+15% Reels); Reels +22% engagement vs Stories; Reels CPM $4-7 vs Stories $6-8 (Reels mais barato); Reels alcance 726M vs Stories menor; Reels lider em discovery (cold audience). **Quando Stories ainda valem**: (a) Retargeting website visitors / IG engagers (warm audience); (b) Time-sensitive offers com countdown sticker; (c) Polls + Quizzes para engagement; (d) Customers existentes (LTV nurture). **Alocacao $5k/mes recomendada moda BR**: **70% Reels Ads** ($3.500): 60-90s creative com sound-on (audio essencial moda — música + voiceover OOTD), discovery campaign Advantage+ Shopping, lookalike 1-3% best customers, multiple creatives 4-8 variations; **20% Stories Ads** ($1.000): retargeting website 30 dias + IG engagers 60 dias, oferta time-sensitive, link sticker para produto, captions obrigatorias; **10% Carousel Reels NOVO** ($500): formato hibrido com **4x performance** que poucos brands usam — testar OOTD swipe, before/after styling, lookbook. **Setup tecnico**: CAPI obrigatorio + catalogo Shopify sync + Aggregated Events priority Purchase, AddToCart, ViewContent, Lead. **Esperado em 90 dias** (BR moda + $5k/mes): 200-400 sales/mes, ROAS 3.5-5x, CPM medio $5-8 (Tier 3 BR vantagem). **NAO**: investir em Feed single image (formato em decline -24% engagement YoY), forcar Stories em audience cold (ROAS metade Reels), pular CAPI (tracking 50% degradado).
